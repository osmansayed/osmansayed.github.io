📢 إشعارات تلقائية للكلاس روم

لما ترفع مادة → الطلاب ياخدوا إيميل فوري!

1 أنشئ Google Apps Script

  1. اذهب script.google.com
  2. اضغط New project
  3. احذف الكود القديم
  4. انسخ من `classroom-notifier.gs` ← Paste هنا
  5. Save → Project name: "Classroom Notifier"

2 ربط بالكلاس روم

  1. اذهب classroom.google.com → فصولك → الفصل
  2. Settings ⚙️ → Class details → Apps Script
  3. Edit project → يفتح الـ Script اللي عملته

3 شغّل الـ Trigger (السحر ✨)

  1. Triggers (الساعة 👈)
  2. Add Trigger
  3. Choose `onCourseWorkCreate`
  4. Save → Allow permissions

4 اختبر

في Apps Script → Functions → `testNotification` → Run → شوف الـ Logs ✅

5 جاهز! 🎉
