## 🚀 البدء السريع

### الطريقة 1️⃣: التشغيل المحلي (على جهازك)

#### Windows
```powershell
# انتقل إلى مجلد المشروع
cd d:\protofilio

# استخدم Python HTTP Server
python -m http.server 8000

# أو استخدم Node.js
npx http-server
```

ثم افتح المتصفح على: `http://localhost:8000`

#### Mac/Linux
```bash
cd /path/to/protofilio

# استخدم Python
python3 -m http.server 8000

# أو استخدم Node.js
npx http-server
```

ثم افتح: `http://localhost:8000`

### الطريقة 2️⃣: نشر على GitHub Pages (مجاني)

#### الخطوة 1: إنشاء Repository
1. اذهب إلى [github.com](https://github.com) وسجل حساب (أو سجل دخول)
2. اضغط **New Repository**
3. اسم الـ Repository: `your-username.github.io`
4. اختر **Public**
5. اضغط **Create Repository**

#### الخطوة 2: رفع الملفات
1. انسخ جميع الملفات من `d:\protofilio` إلى مجلد المشروع
2. حرك جميع الملفات إلى الجذر (root) من المشروع
3. استخدم Git لرفع الملفات:

```bash
# انتقل إلى المشروع
cd your-local-folder

# هيّئ Git
git init
git add .
git commit -m "Initial commit - Portfolio & Analytics Assistant"
git branch -M main
git remote add origin https://github.com/YOUR-USERNAME/your-username.github.io.git
git push -u origin main
```

#### الخطوة 3: تفعيل GitHub Pages
1. اذهب إلى Settings في الـ Repository
2. اختر **Pages** من اليسار
3. انتها! الموقع جاهز على: `https://YOUR-USERNAME.github.io`

## 💡 نصائح الاستخدام

### 📊 استخدام أداة تحليل البيانات

1. **رفع الملف:**
   - اضغط على منطقة الرفع أو اسحب الملف مباشرة
   - الملفات المدعومة: CSV, XLSX, XLS

2. **عرض البيانات:**
   - التبويب "الرسوم البيانية" - عرض بصري للبيانات
   - التبويب "الجداول" - عرض البيانات الكاملة
   - التبويب "الإحصائيات" - حسابات إحصائية

3. **الدردشة:**
   - اسأل أي سؤال عن تحليل البيانات
   - استخدم الأسئلة المقترحة كنقطة بداية
   - النظام يعطيك نصائح احترافية
